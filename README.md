# windmill-codegen
Generating code for Windmill integrations

# Installation

1. Create a virtual environment
```bash
 python -m virtualenv codegen
```


2. Activate it
```bash
 source codegen/bin/activate
```

3. Install the dependencies
```bash
 pip install -r requirements.txt
```

4. Run the CLI
```bash
 python codegen.py
```
